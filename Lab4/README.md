# Лабораторная 4. Машинное обучение с учителем. Методы регрессии

## 1. Описание

В данной лабораторной работе необходимо изучить на практике методы data mining для решения задач регрессии на основе датасетов по диабету и по игре League of Legends


Программа работает с датасетом Diabet и по игре **League of Legends** — **League of Legends Ranked Matches**.

Датасет Diabet доступен по следующей ссылке:  

[Diabet - Kaggle](https://www.kaggle.com/datasets/mathchi/diabetes-data-set)

Датасет LoL доступен по следующей ссылке:  
[League of Legends Ranked Matches - Kaggle](https://www.kaggle.com/datasets/paololol/league-of-legends-ranked-matches)

## 2. Запуск

Для запуска необходимо положить датасеты в папку data вне проекта

Для запуска интерфейса - запустить файл main.py

## 3. Требования

Установите зависимости с помощью команды в корне всего проекта:
