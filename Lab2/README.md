# Лабораторная 2. Исследовательский анализ данных. Постановка гипотез Категориальные данные

## 1. Описание

В данной лабораторной работе необходимо произвести разведочный анализ двух датасетов, по ним составить гипотезы и их проанализировать, проанализировать корреляцию между признаками, а также реализовать алгоритм градиентного спуска.


Программа работает с датасетом mpg (нативно доступен в библиотеке seaborn, устанавливать отдельно его не нужно) и по игре **League of Legends** — **League of Legends Ranked Matches**.

Датасет доступен по следующей ссылке:  
[League of Legends Ranked Matches - Kaggle](https://www.kaggle.com/datasets/paololol/league-of-legends-ranked-matches)

## 2. Запуск

Для запуска анализа датасета mpg необходимо запустить файл commop.py.

Для запуска анализа датасета по игре необходимо запустить файл self_work.py

## 3. Требования

Установите зависимости с помощью команды в корне всего проекта:
```text
pip install -r requirements.txt
```
