# Лабораторная 5. Машинное обучение без учителя. Методы снижения размерности и задача кластеризации


## 1. Описание

В данной лабораторной работе изучить на практике методов data mining для снижения размерности данных и решения задач кластеризации с использованием Python (scikit-learn, umap-learn, joblib). 

Программа работает с датасетом Penguins и по игре **League of Legends** — **League of Legends Ranked Matches**.

Датасет Penguins доступен по следующей ссылке:  

[Penguins - Kaggle](https://www.kaggle.com/datasets/youssefaboelwafa/clustering-penguins-species)

Датасет LoL доступен по следующей ссылке:  
[League of Legends Ranked Matches - Kaggle](https://www.kaggle.com/datasets/paololol/league-of-legends-ranked-matches)

## 2. Запуск

Для запуска необходимо положить датасеты в папку data вне проекта

Для запуска интерфейса - запустить файл main.py

## 3. Требования

Установите зависимости с помощью команды в корне всего проекта:
