# Лабораторная 3. Бинарная и многоклассовая классификация. Оценка качества задачи классификации

## 1. Описание

В данной лабораторной работе необходимо изучить и применить на практике методы бинарной классификации для решения задачи диагностики злокачественных новообразований на основе датасета Breast Cancer Wisconsin, League of Legends а также выполнить оценку качества построенных моделей машинного обучения.


Программа работает с датасетом Breast Cancer Wisconsin (нативно доступен в библиотеке seaborn, устанавливать отдельно его не нужно) и по игре **League of Legends** — **League of Legends Ranked Matches**.

Датасет доступен по следующей ссылке:  
[League of Legends Ranked Matches - Kaggle](https://www.kaggle.com/datasets/paololol/league-of-legends-ranked-matches)

## 2. Запуск

Для запуска анализа датасета Breast Cancer Wisconsin необходимо запустить файл common.py.

Для запуска анализа датасета по игре необходимо запустить файл self_work.py

## 3. Требования

Установите зависимости с помощью команды в корне всего проекта:
```text
pip install -r requirements.txt
```
